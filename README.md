# LiteHouse – Smart Home Control System

## Overview
LiteHouse is a smart home control system developed using the STM32F103C8T6 microcontroller. 
The system enables wireless control of home appliances (lights, fans, etc.) via Bluetooth using the HC-06 module.   

The project focuses on safety, reliability, and expandability for future IoT integration. 

## Hardware Used
- MCU: STM32F103C8T6 (ARM Cortex-M3) 
- Bluetooth Module: HC-06 (UART communication) 
- MQ2 Gas sensor, single led, servo 
- LCD 16x02 via GPIO 4 bit (driver HD44780) 
- Power supply: 5V regulator (source input 8.4V) 

## Software Environment
- Language: C 
- Development Tool: STM32CubeIDE 
- HAL Library (STM32 Hardware Abstraction Layer) 
- Bluetooth communication via UART protocol 
- Control App: MIT APP and custom GUI in MIT APP INVENTOR 

## Project Structure
```
LiteHouse/
├─ Core/                      # Main application code auto-generated by CubeMX
├─ Drivers/                   # HAL drivers and middleware
├─ Debug/                     # Build output files (generated after compiling)
├─ .settings/                 # Project settings for STM32CubeIDE
├─ .cproject                  # IDE configuration file
├─ .mxproject                 # STM32CubeMX project metadata
├─ .project                   # STM32CubeIDE project definition
├─ lithub Debug.launch        # Debugger configuration file for IDE
├─ lithome.ioc                # CubeMX configuration file (pinouts, clocks...)
├─ STM32F103C8TX_FLASH.ld     # Linker script for memory mapping
├─ smart_house2.aia           # Android Bluetooth App project (MIT App Inventor)
└─ README.md                  # Documentation

```

## Getting Started
1. Flash firmware to STM32F103C8T6 using ST-Link 
2. Power hardware (USB or regulated adapter) 
3. Pair phone with **HC-06** (PIN: 1234 / 0000) 
4. Send command via Bluetooth app to control devices
